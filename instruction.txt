## Setup

### 1 Install Ollama
https://ollama.com/download

### 2 Pull model

ollama pull llama3

### 3 Clone repo

git clone https://github.com/im11z/multi-agent-ai-tutor.git
cd multi-agent-ai-tutor

### 4 Create virtual environment

python -m venv venv

Activate

Windows:
venv\Scripts\activate

### 5 Install dependencies

pip install -r requirements.txt

### 6 Run project

python main.py
