# Multi-Agent AI Tutor

A local AI tutoring system built using Python and Ollama.

## Features
- Tutor agent explains topics
- Quiz agent generates questions
- Evaluator agent checks answers
- Feedback agent provides guidance
- Progress tracking

## Tech Stack
- Python
- Ollama
- Local LLM (Llama3)

## Setup

### Install Ollama
Download: https://ollama.com/download

Pull model:
ollama pull llama3

### Install Python dependencies
pip install -r requirements.txt

### Run the tutor
python main.py
